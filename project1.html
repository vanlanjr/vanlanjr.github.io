Project 1: Note Tracker

The Problem

Throughout college, I've been using the Notes app on my iPhone to keep track of all of my assignments in my classes. This has been a very manual process. Every time I get a new assignment, I need to put it in the proper section for the given section. Once I complete an assignment, I delete it from the note. For some assignments, I decide a priority by putting a certain number of exclamation points next to it if it needs to be completed soon. I would like to create an app that would make this whole process easier and more efficient.

The Idea

My idea is to write an application using Vue.js that allows the user to create a Note for tracking a List of Tasks that needs to be completed. The applcation will allow for the user to create multiple notes, just like the iPhone version.

Draft of First Requirements

  • The application allows the user to create a note or multiple notes
  • The user is allowed to create a checklist of tasks on a note
  • The tasks have a name, description, due date, and priority
  • The application reminds the user when a task will be due soon
  • The user can view previously completed tasks on the notes

After More Consideration...

After more consideration, I realized that the one functionality that I really want, in order to get the most value out of the app I create, is for the application to send push notifications to my mobile device when it is installed as a native application. The reason I want this is because that is a key feature that is missing from the plain jane Notes app on my iPhone and Macbook.
